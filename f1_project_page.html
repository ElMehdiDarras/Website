F1 Race Strategy Analysis WORK IN PROGRESS

Formula 1 strategy analysis tool studying the impact of pit stops and tire wear, this is a passion project of mine so am taking my time to perfect it ":)" .

Project Overview

The F1 Race Strategy Analysis project is an ongoing data science initiative that dives deep into Formula 1 race strategies to understand the critical factors that determine race outcomes. This analytical tool examines the complex relationship between pit stop timing, tire degradation patterns, and overall race performance.

Using real Formula 1 session data obtained through the FastF1 API, this project applies statistical analysis and data visualization techniques to uncover strategic insights that could help understand why certain pit strategies succeed while others fail. The analysis focuses on tire compound effectiveness, optimal pit windows, and the impact of strategic decisions on final race positions.

Team & Collaborators

Solo Developer & Data Analyst

Sole developer responsible for the entire project implementation, including data extraction from the FastF1 API, statistical analysis of race strategies, and creation of insightful visualizations. Designed and developed custom algorithms to analyze pit stop effectiveness and tire performance patterns across multiple F1 seasons.

Key Features

Race Data Analysis

Comprehensive processing of real-time Formula 1 session data including lap times, sector times, pit stop duration, and tire information. The system extracts and processes data from multiple race weekends to build a comprehensive dataset for strategy analysis.

Pit Stop Strategy Evaluation

In-depth analysis of pit stop timing and its impact on race results. The system evaluates different pit strategies by comparing lap time degradation, track position changes, and final race outcomes to identify optimal pit windows for various race scenarios.

Tire Performance Tracking

Detailed analysis of tire degradation patterns across different compounds (soft, medium, hard) and track conditions. The system tracks performance drop-off rates and identifies the most effective tire strategies for different circuit characteristics.

Interactive Visualizations

Custom charts and graphs created with Matplotlib that visualize strategy comparisons, lap time evolution, tire performance curves, and pit stop effectiveness. These visualizations make complex race data accessible and provide clear insights into strategic decision-making.

  • Data Source: FastF1 API providing official Formula 1 timing and telemetry data from race sessions, practice, and qualifying
  • Data Processing: Pandas-based data cleaning and transformation pipelines to structure raw F1 data for analysis
  • Analysis Engine: Custom Python algorithms for statistical analysis of pit strategies, tire performance, and race outcomes
  • Visualization: Matplotlib-powered charts and graphs for data visualization and insight presentation
  • Development Environment: Jupyter Notebooks for interactive data exploration and iterative analysis development
